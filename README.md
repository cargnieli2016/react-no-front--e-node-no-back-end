# 🚀 Jornada Fullstack: Do Zero ao React & Node

Este repositório é o meu "diário de bordo" no mundo do desenvolvimento web. Aqui estou documentando todo o meu aprendizado, começando pelas bases sólidas de HTML e CSS, avançando para o Front-end com React e, futuramente, explorando o Back-end com Node.js.

---

## 📂 Estrutura do Projeto

O projeto está organizado para facilitar a revisão dos conceitos aprendidos em cada etapa:

### 🏗️ [Front-end](./Front-end)
Nesta pasta, explorei os pilares da web:

#### **HTML (HyperText Markup Language)**
1.  **Estrutura Básica:** Entendendo a anatomia de um documento HTML.
2.  **Meta Tags:** Importância para SEO e responsividade.
3.  **Tags Essenciais:** Cabeçalhos, parágrafos, listas e muito mais.
4.  **Atributos e Mídia:** Como trabalhar com links e imagens.
5.  **Tabelas e Semântica:** Criando sites acessíveis e bem estruturados.

#### **CSS (Cascading Style Sheets)**
1.  **Cores e Fontes:** Dando vida e estilo ao texto.
2.  **Métodos de Inserção:** CSS Inline, Interno e Externo.
3.  **Box Model:** Entendendo margens, bordas e preenchimentos.
4.  **Display:** O comportamento dos elementos na página.
5.  **Flexbox 🧩:** Onde a mágica do layout acontece (Direction, Justify, Align, Wrap).

## 🚀 Como ver os resultados?

Para visualizar qualquer uma das aulas, você pode:
1.  Navegar até a pasta desejada (ex: `Front-end/CSS/6 - flex-box/1 - flex-direction`).
2.  Abrir o arquivo `index.html` diretamente no seu navegador.
3.  **Dica de Ouro:** Se você usa o VS Code, recomendo a extensão **Live Server** para ver as mudanças em tempo real enquanto digita o código!
